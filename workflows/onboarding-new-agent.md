# Workflows

Documented processes for agent collaboration.

## Onboarding a New Agent

1. Add agent to [agents.yaml](../agents.yaml)
2. Create `memory/agents/<name>/` directory
3. Share this repo URL with the agent
4. Agent clones and reads `memory/shared/` on first run
